HTML이란 무엇인가?

Hypertext Markup Language(HTML) is the standard markup language for creating web pages and web applications.web browsers receive HTML documents from a web server or from local storage and render them multimedia web pages. HTML describes the structure of a web pages semantically and origianlly included cues for the appearance of the document.

HTML elements are the building blocks of HTML pages. With HTML constructs, images and other objects, such as interactive forms, may be embedded into the redered page. It provides a means to create structured documets by denoting structural semantics for text such as headings, paragraphs, lists, links, quotes and otehr items. HTML elements are delineated by tags, written using angle brackets.
